Step 1 - Start everything

Bring the stack up with Docker Compose and wait until the five services are ready.

Run
make up
Look for
All containers healthy in Docker
Why it matters
Nothing else makes sense until the stack is live

Step 2 - Check the health endpoints

Verify the system is actually alive before you trust any routing or rollout signal.

Run
curl /health on 8080, 8081, 8082, 8083, and 8084
Look for
status: UP everywhere
Note
Rollout controller now exposes /health for the checklist

Step 3 - Confirm the flag state

The rollout starts dark. The feature flag should not expose canary traffic until you intentionally change it.

Run
